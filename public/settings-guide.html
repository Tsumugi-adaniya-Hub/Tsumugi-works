外部サービス接続設定

アプリのルートにある .env ファイルを編集して設定します。変更後は npm start を再起動してください。

1. Notion（タスク連携）

  1. https://www.notion.so/my-integrations を開く
  2. 「+ 新しいインテグレーション」をクリック
  3. 名前を入力（例: つむぎワークス管理アプリ）→ 送信
  4. 「シークレット」欄の secret_xxxx をコピー
  5. Notionで連携したいタスクDBを開き、右上「…」→「接続」→ 作成したインテグレーションを追加
  6. DBページのURLから ID を取得（例: notion.so/xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx?v=... の xxx...xxx 部分32文字）
NOTION_TOKEN=secret_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
NOTION_TASKS_DB_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

2. Google カレンダー

  1. Google Cloud Console を開く → プロジェクトを作成（または既存を選択）
  2. 「APIとサービス」→「ライブラリ」→「Google Calendar API」を有効化
  3. 「APIとサービス」→「認証情報」→「認証情報を作成」→「OAuth 2.0 クライアント ID」
  4. アプリケーションの種類: ウェブアプリケーション
  5. 承認済みのリダイレクト URI に http://localhost:3000/auth/google/callback を追加
  6. クライアントID・クライアントシークレットをコピー
GOOGLE_CLIENT_ID=xxxxxxxx.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=GOCSPX-xxxxxxxxxx
設定後、npm start で起動してから http://localhost:3000/auth/google を開いてGoogleアカウントで認証してください。
認証は初回のみ必要です（トークンはアプリ内に保存されます）。
